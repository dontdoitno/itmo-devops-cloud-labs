## 2. Описание Сервисов AWS
 > Amazon Web Services (AWS) – это самое распространенное в мире облако с широчайшими возможностями, предоставляющее более 200 полнофункциональных сервисов для центров обработки данных во всем мире.
|Amazon Application Stream|Полностью управляемый безопасный сервис потоковой передачи приложений для рабочего стола из облака AWS на любое устройство. Переписывать приложения не требуется. А вот пользователи AppStream 2.0 получают мгновенный и гибкий доступ к нужным приложениям через браузер, поддерживающий HTML5.|
|Amazon Athena|Бессерверный интерактивный аналитический сервис, построенный на базе фреймворков с открытым исходным кодом и поддерживающий форматы открытых таблиц и файлов. Athena предоставляет упрощенный и гибкий способ анализа петабайтов данных там, где они находятся. |
|Amazon Cloud Directory|Специальный облачный сервис многопользовательских каталогов с высокой производительностью и широкими возможностями масштабирования. Он предоставляет в распоряжение пользователя веб-каталоги, которые позволяют легко организовывать ресурсы приложений (пользователей, группы, местоположения, устройства и политики), обеспечить управление ими и поддерживать сложные внутренние связи.|
|Amazon Document Database|Полностью управляемая база данных документов с поддержкой JSON, которая упрощает эксплуатацию важнейших рабочих нагрузок по обработке документов практически в любых масштабах без необходимости в управлении инфраструктурой. |
|Amazon Elastic Inference|Сервис AWS, который позволяет добавлять недорогие вычислительные мощности для ускорения выполнения задач машинного обучения.|
|AWS IoT|Помогает подключаться к миллиардам устройств и управлять ими. способствует хранению и анализу данных IoT для рабочих нагрузок в промышленности, сфере потребительских товаров, коммерческой и автомобильной отраслях.|
|IoT Device Management|Упрощает управление IoT-устройствами, позволяя вам управлять и контролировать весь жизненный цикл устройств и датчиков в одном месте — от планирования и подключения до мониторинга и обслуживания, а затем до их вывода из эксплуатации.|
|AWS Code Pipeline|Сервис непрерывной доставки, который позволяет моделировать, визуализировать и автоматизировать действия, необходимые для выпуска программного обеспечения. С помощью AWS CodePipeline можно моделировать весь процесс выпуска, от сборки кода, развертывания в промежуточной среде и тестирования приложения до запуска в рабочей среде.|
|AWS X‑Ray|Сервис, который помогает разработчикам анализировать распределенные приложения в рабочей среде с помощью комплексного отслеживания. Его можно использовать для поиска проблем с производительностью, ошибок в пограничных ситуациях и других трудноопределяемых проблем. |
|CodeBuild|Полностью управляемый сервис сборки в облаке. CodeBuild компилирует ваш исходный код, запускает модульные тесты и создает артефакты, готовые к развертыванию . CodeBuild устраняет необходимость в предоставлении, управлении и масштабировании собственных серверов сборки.|
|Amazon Machine Learning|Позволяют персонализировать рекомендации, обнаруживать мошенничество, управлять запасами, оптимизировать цепочку поставок и улучшать обслуживание клиентов .|
|Amazon Polly|Предоставляет API, который позволяет быстро интегрировать в приложение возможность синтеза речи. Достаточно отправить в API Amazon Polly текст, который необходимо преобразовать в речь, и сервис сразу же направит обратно в приложение соответствующий аудиопоток. Таким образом, можно немедленно начать потоковую передачу или сохранить аудиопоток в стандартном аудиоформате, например MP3.|
|Amazon Personalize|Помогает повысить качество обслуживания клиентов с помощью персонализации на основе искусственного интеллекта. С помощью механизма рекомендаций Amazon Personalize можно сделать взаимодействие с пользователями гиперперсонализированным в реальном времени в больших масштабах, чтобы повысить вовлеченность пользователей, лояльность клиентов и бизнес-результаты.|
|Amazon Rekognition|Представляет собой сервис для распознавания изображений на базе глубокого обучения. Он способен распознавать объекты, сцены и лица, извлекать текст, распознавать знаменитостей и выявлять недопустимый контент на изображениях. Этот сервис также дает возможность искать и сопоставлять лица. Rekognition Image основан на проверенной технологии глубокого обучения с возможностью масштабирования, разработанной экспертами Amazon в области компьютерного зрения. |
|Amazon Elastic Compute Cloud (EC2) Веб-сервис, который предоставляет безопасную, масштабируемую вычислительную мощность в облачке.|
